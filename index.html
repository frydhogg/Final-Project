Intro to html

HTML atau Hyper Text Markup Language adalah sebuah bahasa markup untuk mendeksripsikan sebuah struktur dan konten dari dokumen web. Dokumen web sering kali kita sebagai pengguna internet kenal sebagai web page. Misalnya kita membuka hacktiv8. Kita akan diberikan sebuah tampilan halaman web. Ini adalah sebuah dokumen yang terbuat dari tag-tag markup dan konten teks yang saling nested satu sama lain.

HTML

Kebanyakan tag HTML biasanya berpasangan seperti <div> dan </div>, tapi beberapa tidak seperti <br/>. Tag pertama biasanya disebut tag awal atau pembuka, lalu tag kedua disebut tag akhir atau penutup.

CSS

CSS (Cascading Style Sheet) adalah file yang memungkinkanmu untuk memisahkan konten HTML dengan style nya. Style sendiri adalah bentuk-bentuk pemercantik atau yang m emperindah konten HTML. Dengan CSS, file HTML kita hanya digunakan untuk konten.

./Lesson 2

JavaScript adalah sebuah bahasa pemrograman yang digunakan untuk membuat sebuah halaman web menjadi interaktif. JavaScript berjalan atau beroperasi di browser yang ada di tiap komputer yang digunakan oleh visitor atau user web kita.

Data Types

Saat kita ingin membuat sebuah program, simple nya adalah program itu pasti terdiri atau tersusun dari bermacam-macam data

Alert

Alert adalah sebuah perintah untuk memberikan popup browser sederhana kepada user. Contohnya seperti sintaks alert("hello")

Prompt

Mirip dengan alert, prompt juga memberikan popup sederhana kepada user, namun juga meminta feedback atau input dari user.

Log

Logging tidak dapat dengan langsung kita lihat hasilnya dengan kasat mata. Log berarti memunculkan pesan tapi dibelakang tampilan.

Variable

Nilai-nilai yang kita buat atau butuhkan di program dapat kita simpan ke dalam sebuah penampung. Penampung ini biasa disebut dengan variable.

Function

Function adalah sebuah blok kode yang disusun untuk menjalankan sebuah tindakan. Blok kode ini dibuat untuk dapat bisa digunakan kembali.

Advance Javascript mempelajari kombinasi antara Function, Array dan Object di JavaScript. Berikut ditampilkan definisi dan kegunaan pada Function, Array dan Object. Function, Array dan Object umumnya digunakan untuk mengolah data dan menyimpan data. Dapat digunakan dari data internal maupun eksternal

  • Function
  • Array
  • Object

Function

Function adalah sebuah blok kode yang disusun sedemikian rupa untuk menjalankan sebuah tindakan. Blok kode ini dibuat untuk dapat bisa digunakan kembali. Kita juga dapat mengirimkan nilai ke dalam sebuah function dengan mencantumkannya ke dalam tanda kurung dalam penulisan function. Untuk mengirimkan nilai lebih dari satu, gunakan tanda , sebagai pemisah.

Array

Array adalah kumpulan dari berbagai data. Kita bisa tulis dengan kurung kotak ( square brackets ) dan butir-butirnya dipisah dengan koma. Indeks array berbasis-nol yang berarti urutan awalnya merupakan 0, keduanya 1, dan seterusnya. Kita juga bisa memasukkan ( insert ), memperbarui/mengubah ( update/change ), atau bahkan meniadakan ( undefine ) nilai yang ada di dalam array. Spesifik di JS, kita bisa gunakan beberapa tipe data berbeda di dalam suatu array. Bahkan memasukkan array ke dalam array

Object

JavaScript merupakan bahasa pemrograman yang berbasis simple-object (Objek sederhana). Objek adalah kumpulan tidak berurut yang merangkai beberapa property dan property memiliki nama/key dan value ( key-value pairs ). Objek dalam JavaScript, sama seperti banyak bahasa pemrograman lainnya, bisa dibandingkan dengan objek dalam kehidupan nyata. Untuk membuat sebuah object literal bisa dengan cara menuliskan kurung kurawal ( curly braces ) kemudian menuliskan nama property yang harus memiliki keyName dan value.

jQuery dan Bootstrap

jQuery adalah sebuah library JavaScript yang cepat, ringan, dan memiliki banyak fitur yang bermanfaat. Library JavaScript adalah kumpulan kode JavaScript yang dibuat oleh orang lain yang dapat kita gunakan dengan mudah dan gratis. jQuery memudahkan kita untuk melakukan penjelajahan dokumen HTML dan memanipulasinya, mengatur efek dan animasi, dan hal-hal keren lainnya, dan tentunya berfungsi di browser manapun.

Bootstrap tidak hanya merupakan framework CSS. Bootstrap juga memiliki kode JavaScript untuk membuat beberapa efek tampilan yang sering digunakan, seperti modal, dan lain-lain.
